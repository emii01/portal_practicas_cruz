Carrera Analista de Sistemas

Este portal permite gestionar las prácticas profesionalizantes de los estudiantes de la carrera de Analista de Sistemas.

Los alumnos desarrollan conocimientos sobre análisis de sistemas, bases de datos y desarrollo web.

Tecnologías utilizadas

Python
Lenguaje utilizado para backend y automatización.
HTML5
Lenguaje de marcado para crear páginas web.
SQL
Lenguaje utilizado para bases de datos relacionales.
